# DPS: A Post-Quantum Proxy Signature Scheme from Dilithium for IoT Applications
A prototype implementation of DPS:A Post-Quantum Proxy Signature Scheme from Dilithium for resource-constrained IoT devices.
## Dependencies
1. SageMath: https://www.sagemath.org/
2. Kyber: `sage -pip install kyber_py`

## Test
Run the DPS.sage 

`sage DPS.sage`
